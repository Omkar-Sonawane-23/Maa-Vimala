How To Add This on a New Tab

  1. Clone the repository: 
    git clone https://github.com/Web403/Maa-Vimala
  2. Navigate to the project directory and locate the index.html file: 
    path/to/Maa-Vimala/index.html
  3. Set this file as your custom homepage in your browser:
    • Go to Browser Settings
    • Search for "Home page"
    • Set to "Custom URL"
    • Paste the path to the index.html file
